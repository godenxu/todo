# 待办管理（Todo）

一个**零依赖、单文件**的本地待办事项管理工具。整个应用就是一个 HTML 文件，用浏览器打开即可使用，数据保存在浏览器本地（localStorage），无需服务器、无需安装、不联网。

适合个人或小团队在内网环境下做轻量的任务跟踪与分工管理。

## ✨ 特性

- **单文件即用**：双击 `todo_*.html` 用浏览器打开就能用，无需任何环境或依赖。
- **快捷输入**：顶部输入框一行话录入任务，支持分类、分工、优先级、到期日的简写语法。
- **表格化管理**：任务、分类、分工、优先级、到期日、来源、自定义共 7 个字段，列可**排序、拖拽换位、拖动改宽**。
- **行内编辑**：双击单元格即可修改；优先级点击循环切换；到期日内置日历选择器。
- **多维筛选**：左侧栏按状态、分类、分工、优先级、来源、自定义标签快速过滤，并实时显示各项数量。
- **软删除 + 撤销**：删除进入"已删除"视图可恢复；新建、完成、删除均可 `Ctrl+Z` 撤销。
- **CSV 导入导出**：一键导出全部数据备份，或从 CSV 批量导入（按 id 更新或新增）。
- **本地优先**：所有数据仅存于本机浏览器，隐私可控。

## 🚀 快速开始

1. 下载仓库中的 `todo_*.html`（如 `todo_202606241227.html`）。
2. 双击用浏览器（推荐 Chrome / Edge）打开。
3. 在顶部输入框输入任务，回车即可创建。

> 首次打开会自动生成几条示例任务，删除即可。

## ⌨️ 快捷输入语法

在顶部输入框中，用空格分隔的标记可以一次性设置任务属性：

| 语法 | 含义 | 示例 |
|------|------|------|
| `#分类` | 设置分类 | `#工作` |
| `@人员` | 添加分工（可多个） | `@张三 @李四` |
| `!1` `!2` `!3` | 优先级（1 高 / 2 中 / 3 低） | `!1` |
| 日期标记 | 设置到期日 | 见下表 |
| 其余文字 | 作为任务标题 | |

**日期标记**支持：

| 写法 | 含义 |
|------|------|
| `today` | 今天 |
| `tomorrow` | 明天 |
| `+3` 或 `+3d` | N 天后 |
| `mon`/`tue`/…/`sun` | 本周/下周最近的星期几 |
| `next-mon` | 下周一 |
| `2026-06-30` | 指定日期 |

示例：

```
整理Q2项目文档 #工作 @张三 @李四 !1 fri
```

> 创建一个标题为「整理Q2项目文档」、分类「工作」、分工张三和李四、优先级高、本周五到期的任务。

## 🔑 键盘快捷键

| 按键 | 功能 |
|------|------|
| `N` | 聚焦到新建输入框 |
| `/` | 聚焦到搜索框 |
| `Ctrl + S` | 导出 CSV |
| `Ctrl + Z` | 撤销上一步（新建 / 完成 / 删除） |
| `Esc` | 清空输入框 / 关闭弹层 |

## 🗂️ 视图与字段

**状态视图**：全部待办、今日到期（含逾期）、本周、已完成、已删除。

**字段说明**：

- **任务**：标题，双击编辑。
- **分类 / 来源 / 自定义**：单选标签，双击从下拉选择或新建。
- **分工**：可多选人员，带序号显示。
- **优先级**：高 / 中 / 低，点击徽章循环切换。
- **到期日**：内置日历与「今天 / 明天 / 下周一 / 清除」快捷选项；逾期红色高亮，今天到期橙色高亮。

## 💾 数据与备份

- 数据保存在浏览器 **localStorage**（键名 `todo_v2`），仅存于本机。
- **更换浏览器、清理浏览器数据会导致数据丢失**，请定期用「导出 CSV」备份。
- 导出的 CSV 为 UTF-8（带 BOM），可用 Excel 直接打开；也可编辑后重新导入。

## 🛠️ 技术栈

- 原生 **HTML + CSS + JavaScript**，无任何框架或第三方库。
- 全部逻辑、样式、结构内联于单个 HTML 文件中。
